# Publicación en Google Play

1. Verifica disponibilidad comercial y posibles marcas de “Future Breaker”.
2. Crea la app en Play Console con paquete `com.scriptly.futurebreaker`.
3. Prepara icono final, banner, capturas, descripción corta/completa y URL de política de privacidad.
4. Completa Data Safety y clasificación de contenido según las funciones reales.
5. Genera AAB release con firma protegida y Play App Signing.
6. Sube primero a prueba interna; corrige errores, ANR y advertencias.
7. Ejecuta pruebas cerradas exigidas para tu tipo de cuenta antes de producción.

Revisa requisitos vigentes de nivel de API y políticas justo antes de publicar, pues cambian con el tiempo.
