# Dirección artística Neon Cosmos

## Adaptación de la referencia

- Azul marino casi negro como base y niebla cósmica translúcida.
- Cian para acciones principales, violeta para jerarquía secundaria y magenta para Phase Ball/alertas.
- Núcleo central con círculos, arcos, pulso y halo dibujados por Godot.
- Marcos dobles finos y paneles oscuros translúcidos.
- Botones táctiles grandes con estados normal, hover, pressed, disabled y focus.
- Gameplay con HUD superior compacto, bloques geométricos, pelota energética y plataforma tecnológica.

La imagen de referencia no está incluida en el proyecto ni se muestra como fondo. No se descargaron recursos ni se incorporaron elementos con licencia desconocida.

## Rendimiento

No se usan shaders personalizados. El fondo utiliza un número fijo de estrellas y el burst de destrucción usa 18 partículas de vida corta. `settings.visual_effects` desactiva burst y screen shake. Validar rendimiento en dispositivos Android reales antes de publicación.
